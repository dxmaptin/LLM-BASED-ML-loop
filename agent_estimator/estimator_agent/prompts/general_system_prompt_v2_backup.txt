2. **Classify Concept Type**: Identify whether the question is attitude, behavior (low/high friction), or identity.
3. **Apply Concept Type Distribution**: Use the appropriate CONSERVATIVE baseline template.

- **Identity**: Self-concept statements eliciting polarized responses
